# Bashly

Private event management SaaS for Sweet 16s, Quinceañeras, Graduations, Anniversaries, and Reunions.

**Stack:** Next.js 14 · Expo 52 · Better Auth · Drizzle ORM · Neon PostgreSQL · Tailwind CSS · shadcn/ui · NativeWind · Stripe · Resend · Cloudflare R2 · Mapbox

## Getting started

```bash
npm install
npm run dev
```

Web: http://localhost:3000
Mobile: Expo — press i for iOS, a for Android
